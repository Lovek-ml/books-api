# Books API

Простое API для работы с книгами на Flask и SQLite.

## Возможности

- Получить список книг
- Найти книгу по названию
- Найти книги по автору
- Найти книги по году
- Получить случайную книгу

## Технологии

- Python
- Flask
- SQLite

## Запуск

Установить Flask:

pip install flask

Запустить приложение:

python books.py
